## General Notes
This configuration is for the [LSC Smart Connect Ledstrip 3203632.1](https://i0.wp.com/www.twoenter.nl/blog/wp-content/uploads/2025/02/3202086_8712879164401-111_01_mv21xs1.webp?w=640&ssl=1)
which comes as a colour and white changing LED strip with controller, IR remote and transformer.

![LSC Smart Connect Ledstrip 3203632.1](https://i0.wp.com/www.twoenter.nl/blog/wp-content/uploads/2025/02/3202086_8712879164401-111_01_mv21xs1.webp?w=640&ssl=1)

## GPIO Pinout
| Pin    | Function      |
| ------ | ------------- |
| P16    | Data RGB and CCT leds |
| P8     | Power for LEDs|
| P9     | Button 1      |
| P24     | Button 2     |
| P28     | Button 3     |
| P26    | IR-Receiver   |
| ADC3   | Mic           |
